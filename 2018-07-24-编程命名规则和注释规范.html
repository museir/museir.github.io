    命名规则

    基本原则：以尽可能少的字符清晰、准确的表达尽可能多的信息。

    • 名称只能由数字、字母和下划线组成
    • 不能使用拼音
    • 不能以数字开头
    • 不能使用编程关键字

    变量命名规则

    循环变量/函数参数/局部变量

    首字母小写，后面单词（如果有的话）首字母大写，其余小写，单词间无间隔。

    结构体/枚举

    前面添加范围限定符，一般使用大写的模块名缩写和下划线组成。

    全局变量

    在局部变量前面添加小写的数据类型说明。

    附上数据类型前缀命名规则

    数据类型 前缀 说明
    array a Array Datas
    boolean b BOOL Data
    char c 8-bit Data
    double d 64-bit Float
    float f 32-bit Float
    int i 32-bit Data
    long l 64-bit Data
    short s 8-bit Data
    point p 指针
    function point fp 函数指针
    string str 字符串

    函数命名规则

    所有单词首字母大写，单词间以下划线(_)隔开。

    常量命名规则

    宏定义/只读变量/枚举常数

    所有单词字母大写，单词间以下划线（_）隔开。

    其他命名规则

    工程名/项目名/包/文件名等始终小写，单词间以下划线(_)分割。
