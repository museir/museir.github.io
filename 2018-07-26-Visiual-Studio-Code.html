    使用Markdown

    无需任何插件，VS Code支持编写并实时预览Markdown文件。

    1. 标题

    2. 分隔符

    3. 字体样式

    4. 代码标记

    5. 引用

    6. 链接

    7. 序列

    8. 表格

    语法：VS Code targets the CommonMark Markdown specification using the markdown-it library.

    参考：https://code.visualstudio.com/Docs/languages/markdown

    使用Hexo

    VS Code扩展->搜索vscode-hexo，安装，按F1，输入hexo，根据提示选择命令执行。

    此插件实现了hexo官方所有的命令，比如hexo clean、hexo generate、hexo deploy、hexo server等。
